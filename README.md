# Photo Swipe

Photo Swipe es una aplicación móvil desarrollada en Flutter que te ayuda a organizar y limpiar tu galería de fotos de manera rápida, divertida e inteligente.

## Características principales
- Pantalla de bienvenida moderna y atractiva.
- Solicitud de permisos para acceder a la galería de fotos.
- Interfaz intuitiva para navegar y gestionar tus imágenes.
- Botón de inicio con diseño personalizado.
- Lógica preparada para implementar limpieza inteligente de la galería.

## Instalación y ejecución
1. Clona el repositorio.
2. Instala las dependencias ejecutando:
	```
	flutter pub get
	```
3. Ejecuta la aplicación en un emulador o dispositivo físico:
	```
	flutter run
	```

## Estructura del proyecto
- `lib/main.dart`: Punto de entrada de la app.
- `lib/welcome_screen.dart`: Pantalla de bienvenida.
- `lib/providers/gallery_provider.dart`: Lógica de gestión de galería.
- `assets/`: Imágenes y recursos gráficos.

## Requisitos
- Flutter SDK
- Permisos para acceder a la galería de fotos
